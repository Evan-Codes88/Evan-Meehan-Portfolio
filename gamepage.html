My Life As A Coder Game

This game was drawn, animated and coded entirely by me. It was used to apply for the Coder Academy Diversity Scholarship. It is currently under development to change it from a scholarhsip application into a fun pixel 2D platformaer. :)

Behind the Scenes - Development

I used Godot and their built in language GD Script to develop this game. I found Godot to be an amazing engine that allowed for easy organisation using nodes. It was an extreme learning curve but I really enjoyed making this game.

Behind the Scenes - Aesprite

I used Aesprite, a pixel art program, to draw all of my characters, tilesets and UI. I animated my own sprites and the flower is definately my favourite death animation.
